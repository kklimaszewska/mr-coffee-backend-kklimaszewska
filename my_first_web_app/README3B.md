This project is a simple schedule website.


#### Table of contents

## Requirements

For development, you will need:
* Node.js
* Frameworks/Librarys:
    - express,
    - js-sha256,
    - mustache-express

	
## Instalation
```
    $ `git clone git@github.com:pl-incode/mr-coffee-backend-kklimaszewska.git`
    $ `cd my_first_web_app`
    $ `npm install`
```

## Running
```
    $ `cd ./my_first_web_app`
    $ `npm start` 
    or
    $ `npm run start`
```
