## Requirements

For development, you will need:
* Node.js
* Frameworks/Libraries:
    - express,
    - js-sha256,
    - mustache-express
    - pg

	
## Instalation
```
    $ git clone git@github.com:pl-incode/mr-coffee-backend-kklimaszewska.git
    $ cd my_first_web_app
    $ npm install
```


## Running
```
    $ cd ./my_first_web_app
    $ npm start PGPASSWORD='*************'
    or
    $ npm run start PGPASSWORD='*************'
```
